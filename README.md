# Ayidaah Beauty

The production storefront for Ayidaah Beauty in Rajagiriya, Sri Lanka.

## Local development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

The site is configured for Netlify with `netlify.toml`. Every push to the connected production branch triggers a new production deployment.

- Framework: Next.js
- Build command: `pnpm build`
- Customer checkout: WhatsApp order handoff
- Currency: Sri Lankan rupees (LKR)
